URL: https://www.bonjour-ratp.fr License: Unofficial reverse-engineered spec

Backend-for-Frontend API powering the bonjour-ratp.fr transit information service. Provides journey planning, real-time departures, line disruptions, mobility providers, and regional transit maps for the Ile-de-France region.

Generated from automated reverse-engineering of the website on 2026-04-19.

Note: All endpoints require the x-api-key header. The API uses Kong 3.9.0 as a gateway behind Cloudflare CDN. Direct curl/HTTP requests are blocked by Cloudflare WAF; use a real browser or simulate browser headers.

API versioning is content-type based via vendor-specific Accept headers (e.g., application/vnd.rss.bff.itinerary-stream.v9+json), not URL-based.

itineraries

Journey planning and route search

Search for transit itineraries

Returns step-by-step journey itineraries as Server-Sent Events (SSE). The response is NOT compressed (content-encoding: identity) and contains encoded polyline geometry for route visualization.

Response format: text/event-stream with JSON data events. Each event contains metadata and an array of itineraries.

lines

Line information, disruptions, and timetables

Get current line disruptions/situations

Returns current disruption and crisis situations for transit lines. If lineId is empty or omitted, returns situations for ALL lines.

Authorizations:
ApiKeyHeader
query Parameters
lineId
string
Example: lineId=

Filter by line ID. Empty string returns all lines. Format: LIG:IDFM:{code} (e.g., LIG:IDFM:C01742 for RER A).
